ข้อมูลเกี่ยวกับระบบ

การจองที่จอดรถ

1. เลือกอาคารที่ต้องการจอง (A หรือ B)

2. เลือกช่องจอดที่ว่าง (แสดงเป็นสีเขียว)

3. กรอกชื่อและเบอร์โทรศัพท์

4. กดปุ่ม "จองที่จอดรถ"

5. ระบบจะแสดงการยืนยันการจอง

1. ไปที่ตาราง "การจองล่าสุด"

2. หาการจองที่ต้องการยกเลิก

3. กดปุ่ม "ยกเลิก" สีแดง

4. ยืนยันการยกเลิก

5. ระบบจะปลดล็อกที่จอดรถทันที

การจองจะคงอยู่จนกว่าคุณจะยกเลิกหรือรถจะออกจากที่จอด โดยระบบจะตรวจสอบสถานะแบบเรียลไทม์ผ่านเซ็นเซอร์

ระบบและเทคนิค

1. ตั้งค่า WiFi ในโค้ด Arduino

2. เปลี่ยน IP address ของเซิร์ฟเวอร์

3. อัปโหลดโค้ดลง Arduino

4. ตรวจสอบการเชื่อมต่อในหน้าตั้งค่า

5. ระบบจะรับข้อมูลอัตโนมัติ

1. ตรวจสอบการเชื่อมต่อสายไฟ

2. ตรวจสอบการตั้งค่า pin ในโค้ด

3. ตรวจสอบการเชื่อมต่อ WiFi

4. ดู Serial Monitor สำหรับข้อผิดพลาด

5. รีสตาร์ท Arduino หากจำเป็น

1. ติดตั้ง Node.js และ npm

2. รันคำสั่ง npm install

3. รันคำสั่ง npm start

4. เปิดเบราว์เซอร์ไปที่ localhost:3000

5. ตั้งค่า IP address ใน Arduino

ข้อมูลทั่วไป

ระบบจองที่จอดรถอัจฉริยะเป็นระบบที่ใช้เทคโนโลยี IoT (Internet of Things) ในการตรวจสอบสถานะที่จอดรถแบบเรียลไทม์ และให้ผู้ใช้สามารถจองที่จอดรถผ่านเว็บไซต์ได้

ระบบสามารถรองรับผู้ใช้ได้ไม่จำกัด เนื่องจากใช้ WebSocket สำหรับการอัปเดตแบบเรียลไทม์ และสามารถขยายได้ตามความต้องการ

ระบบจะแจ้งเตือนเมื่อ:

• มีการจองที่จอดรถใหม่

• ระบบตรวจพบความผิดปกติ

• มีการอัปเดตระบบ

• การจองใกล้หมดอายุ
